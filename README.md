## Features
- Shows used / remaining / total quota
- Displays renewal & expiry dates
- Direct data from WE servers

---

## Usage

### 1. Install requirements
```bash
pip install -r requirements.txt
````

### 2. eidt `.env` file
exapmle:
```env
WE_LANDLINE=05XXXXXXXX
WE_PASSWORD=your_password
```

### 3. Run the tool

```bash
python main.py
```

A small window will appear showing your internet quota details.

## Requirements
